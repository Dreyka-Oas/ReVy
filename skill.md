# RÈGLES

- 100 lignes max fichier
- 5 fichiers max dossier
- Pas commentaires
- Réponses FR ultra-courtes, style télégraphique comme homme de cromagnon, tokens min. Sous-agents le plus possible.

# CONVENTIONS

- snake_case fichiers, PascalCase types
- Arrow functions pour callbacks
- async/await pour operations I/O
- #[tauri::command] pour commandes Rust
