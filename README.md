# ReVy

Gestionnaire de versions Minecraft avec Tauri.

## Fonctionnalités

- Détection automatique loaders (Fabric, Forge, NeoForge)
- Gestion Java via SDKMAN
- Lancement de versions
- Test global multi-versions

## Tech

- **Frontend**: Vanilla JS, CSS
- **Backend**: Rust (Tauri)

## Rules

- ≤100 lignes/fichier
- ≤5 fichiers/dossier
- Pas commentaires
- async/await I/O

## Dev

```bash
npm run dev
npm run tauri dev
```

## Build

```bash
npm run build
npm run tauri build
```
